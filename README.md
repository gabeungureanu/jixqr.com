# jixqr.com

Web and API applications for jixqr.com.

## Structure
- `web/` - Web Application
- `api/` - API Application
- `shared/` - Shared code
- `deploy/` - Deployment scripts

## Environments
| Env | Web | API | Branch |
|-----|-----|-----|--------|
| Dev | dev.jixqr.com | dpi.jixqr.com | develop |
| Prod | jixqr.com | api.jixqr.com | main |

## Quick Start
```bash
cd web && npm install && npm run dev
cd api && npm install && npm run dev
```
